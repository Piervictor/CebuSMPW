
  # CartSmart Circuit Web App

  This is a code bundle for CartSmart Circuit Web App. The original project is available at https://www.figma.com/design/7py5KakjC7KEjL8lvHWs1w/CartSmart-Circuit-Web-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Opening With Live Server

  This project is built with Vite, so the source `index.html` depends on Vite to compile `src/main.tsx` and CSS modules.

  If you want to use Live Server:

  1. Run `npm run build`
  2. Open `dist/index.html` with Live Server, not the root `index.html`

  The app now uses hash routing and relative asset paths so the built output works on static servers without SPA fallback configuration.
